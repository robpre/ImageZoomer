# ImageZoomer

ImageZoomer is a lightweight Chrome extension for zooming, panning, rotating and previewing images on webpages.

## Features

- Double-tap Ctrl to open the image under your cursor in fullscreen
- Optional auto-hover preview mode
- Resizable hover preview window
- Remembers your preferred hover preview size
- Mouse wheel zoom
- Cursor-based panning with extra edge buffer
- Press R to rotate images
- Press Escape or click to close fullscreen mode
- Detects images beneath transparent clickable overlays
- Supports normal images, picture elements, CSS background images and many SVG image elements
- Runs locally in your browser
